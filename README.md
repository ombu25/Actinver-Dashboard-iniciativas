### Cómo actualizar el dashboard

1. Descarga `index.html`
2. Abre el archivo en un editor de texto (VS Code, Sublime, etc.)
3. Busca la sección `const CATEGORIES = [` en el JavaScript
4. Actualiza los valores en los arrays `data:` de cada KPI con los datos reales
5. Sube el archivo actualizado a GitHub (reemplaza el anterior)

---

### Cómo hacer el deploy en GitHub Pages

1. Crea un repositorio nuevo en [github.com](https://github.com) llamado `actinver-dashboard`
2. Sube todos los archivos de esta carpeta
3. Ve a **Settings → Pages**
4. En *Branch* selecciona `main` y carpeta `/ (root)` → **Save**
5. Espera 1–2 minutos y tu dashboard estará en vivo
